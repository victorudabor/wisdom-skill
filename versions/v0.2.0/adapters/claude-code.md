# Claude Code Installation

Project scope:

```bash
python3 scripts/install_skill.py --target claude-project --project /path/to/repository
```

Personal scope:

```bash
python3 scripts/install_skill.py --target claude-user
```

Equivalent locations:

- project: `.claude/skills/exact-ui-replica/`
- personal: `~/.claude/skills/exact-ui-replica/`

Invoke explicitly with `/exact-ui-replica` or allow automatic activation through the description.
