# Codex Installation

Repository scope:

```bash
python3 scripts/install_skill.py --target codex-project --project /path/to/repository
```

User scope:

```bash
python3 scripts/install_skill.py --target codex-user
```

Equivalent locations:

- repository: `.agents/skills/exact-ui-replica/`
- user: `~/.agents/skills/exact-ui-replica/`

Invoke explicitly as `$exact-ui-replica` or allow implicit activation through the skill description.
