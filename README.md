# Wisdom Skill

Wisdom Skill is an advanced installable AI coding skill for analysing MVP designs, generating precise implementation specifications, building exact UI replicas, repairing existing implementations, and verifying visual accuracy across compatible AI coding agents.

## Releases

- `versions/v0.1.0` — initial Exact UI Replica skill
- `versions/v0.2.0` — expanded prompt, build, repair, and hybrid modes with validation and visual-difference tooling

Packaged ZIP releases are included at the repository root.

## Current version

The recommended release is **v0.2.0**. See `versions/v0.2.0/README.md` for installation and usage.
